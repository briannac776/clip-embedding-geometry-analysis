# clip-embedding-geometry-analysis
**CLIP Embedding Geometry Analysis
**
This repository contains the code and processed data used for the analyses presented in the manuscript:

“Geometric structure of similarity in high-dimensional image embeddings: a comparison of cosine and euclidean metrics”

**Project Overview
**
This study investigates the geometric organization of high-dimensional CLIP image embeddings using a controlled subset of the CIFAR-10 dataset. The analyses compare cosine similarity and Euclidean distance through embedding norm analysis, semantic neighborhood evaluation, and dimensionality reduction methods including PCA and UMAP.
